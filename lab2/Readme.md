# Project 1

Project 1 is about creating a client and a server for Calculator app

## Client 1

Client 1 can be started by running the following commands

```
cd client1
npm install 
npm start
```


## Server 1

Server 1 can be started by running the following commands

```
cd server1
npm install 
npm start
```
